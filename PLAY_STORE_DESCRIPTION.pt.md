# Bingo Drawer

Um sorteador de números de bingo simples e acessível. Defina o maior número, sorteie sem repetir e veja cada sorteio destacado em uma grade limpa. Ótimo para jogos rápidos e sessões em grupo.

## Principais recursos
- Escolha seu intervalo: defina o número máximo a sortear.
- Um toque para sortear: sorteia um número único a cada vez até acabar o intervalo.
- Grande revelação: um overlay grande e de alto contraste mostra rapidamente o último número para leitura fácil.
- Visão geral em grade: números sorteados são destacados; os demais continuam visíveis.
- Controle da sessão: reinicie com confirmação para começar uma nova rodada.
- Persistência de estado: seu máximo, sorteios e último número são restaurados após fechar o app.

## Como jogar
1) Digite o maior número desejado (ex.: 70) e toque em **Aplicar**.
2) Toque em **Sortear** para pegar o próximo número. O overlay mostra o resultado; a grade destaca o número sorteado.
3) Toque em **Reiniciar** (e confirme) para limpar a sessão e definir um novo intervalo.

## Acessibilidade
- Overlay grande para o último sorteio, com texto autoajustável.
- Destaque claro na grade para números sorteados.
- Entrada e rótulos compatíveis com tema claro/escuro.

## Notas
- Funciona offline; não requer login.
- Feito para telefones e tablets.
