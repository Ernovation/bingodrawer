# Bingo Drawer

一款简单易用的宾果抽号应用。设置最大号码，抽取时不重复，并在清晰的网格中高亮已抽出的号码。大的覆盖层会短暂显示最新号码，方便阅读。

## 功能
- 设置并应用最大号码；抽取唯一号码直到用完。
- 大号覆盖层显示最新号码，自动适配网格区域。
- 网格高亮已抽出的号码；未抽出的保持可见。
- 重置前需确认；应用后输入框隐藏，重置后再显示。
- 状态在应用重启间持久保存（最大值、已抽号码、最后号码、输入文本）。
- 适配明/暗主题；输入隐藏时键盘也会收起。

## 开始使用
```bash
./gradlew :app:assembleDebug
```
将 `app/build/outputs/apk/debug/` 中的 APK 安装到设备或模拟器。

## 使用步骤
1) 输入最大号码，点击 **Apply**。
2) 点击 **Draw** 获取唯一号码；覆盖层闪现结果，网格高亮它。
3) 点击 **Reset**（确认）清空会话并设置新范围。

## 无障碍
- 大号、高对比度覆盖层，文字自动适配。
- 网格清晰标注已抽号码。
- 主题感知的输入与标签。
